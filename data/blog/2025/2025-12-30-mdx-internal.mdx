MDX file is parsed into an AST, and then transformed into a jsx tree as a component.
https://mdxjs.com/docs/using-mdx/

```mdx
export function Thing() {
  return <>World</>
}

# Hello <Thing />
```

```jsx
/* @jsxRuntime automatic */
/* @jsxImportSource react */

export function Thing() {
  return <>World</>
}

export default function MDXContent() {
  return (
    <h1>
      Hello <Thing />
    </h1>
  )
}
```
